# Simple RAG AI Assistant 🤖
A proof-of-concept AI Engineer project using Python, LangChain, and Groq.

## 🚀 Features
- **Semantic Search:** Uses HuggingFace Embeddings to find relevant data.
- **Context Grounding:** Prevents AI hallucinations by providing local file context.
- **High Performance:** Integrated with Groq LPU for near-instant responses.

## 🛠️ Tech Stack
- **Language:** Python
- **Orchestration:** LangChain
- **Vector DB:** ChromaDB
- **LLM:** Llama 3 (via Groq)

## 📖 What I Learned
- How to turn text into **Vectors** for semantic search.
- Managing **Context Windows** in Large Language Models.
- Setting up a cloud-based AI development environment.
